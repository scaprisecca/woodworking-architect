# Closet Storage — Project Template

## Common Configurations

### Walk-in Closet
Full room with hanging, shelving, and drawer sections on multiple walls.

### Reach-in Closet
Standard closet with doors — maximize storage in limited depth (typically 24").

### Wardrobe / Armoire
Freestanding or built-in unit replacing or supplementing a closet.

## Project-Specific Consultation Questions

### Usage
- [ ] Who uses this closet? (one person, couple, child)
- [ ] What's stored? (clothes, shoes, accessories, linens, seasonal items)
- [ ] How many linear feet of hanging needed? (long hang vs double hang)
- [ ] How many pairs of shoes?
- [ ] Folded items — how many shelves/drawers needed?
- [ ] Accessories storage (ties, belts, jewelry, scarves)?
- [ ] Hamper/laundry basket space needed?
- [ ] Safe or lockable storage needed?
- [ ] Luggage or bulky item storage?

### Preferences
- [ ] Open shelving or drawers for folded clothes?
- [ ] Prefer hanging or folding?
- [ ] Mirror desired (full-length)?
- [ ] Lighting preference (overhead, LED strips, motion-activated)?
- [ ] Valet rod or pull-down hanging rod?

## Measurements Specific to Closets
- [ ] Interior width, depth, and height
- [ ] Door opening width and height
- [ ] Door type (bifold, sliding, hinged — swing direction)
- [ ] Any obstructions (light fixtures, vents, access panels)
- [ ] Floor material (carpet affects base contact)
- [ ] Existing rod and shelf height

## Typical Materials
- **Carcass/shelves:** 3/4" melamine or plywood
- **Drawers:** 1/2" plywood
- **Backing:** 1/4" plywood (or skip if against painted wall)
- **Rods:** 1-5/16" chrome or oval

## Special Considerations
- **Standard hanging heights:** Long hang 66"-68" from floor, double hang upper 80"-84" / lower 40"-42"
- **Shelf depth:** 12"-14" for most clothing, 16" for bulky items
- **Drawer depth:** 6" shallow (accessories), 8"-10" deep (folded clothes)
- **Shoe shelves:** Angled at 15-20 degrees, 8"-10" deep, 6"-7" vertical spacing

## Common Upgrades to Offer
- LED lighting (motion-activated)
- Velvet-lined jewelry drawers
- Pull-down closet rod for high hanging
- Built-in hamper with removable bag
- Valet rod (pull-out)
- Belt/tie rack
- Full-extension drawer slides
- Glass-front drawers
