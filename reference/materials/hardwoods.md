## Buying Lumber

### Board Foot Calculation
**Board Foot (BF) = (Thickness" x Width" x Length") / 144**

Or: **BF = (T x W x L) / 144**

Example: A 4/4 board that's 6" wide and 8' long = (1 x 6 x 96) / 144 = 4 BF

### Rough Lumber Thickness (Quarter System)
| Designation | Rough Thickness | Surfaced (S2S) Thickness |
|-------------|----------------|--------------------------|
| 4/4 | 1" | 3/4" (13/16" typical) |
| 5/4 | 1-1/4" | 1" |
| 6/4 | 1-1/2" | 1-1/4" |
| 8/4 | 2" | 1-3/4" |

### Grades (NHLA)
| Grade | Description | Typical Yield |
|-------|-------------|---------------|
| FAS (First and Seconds) | Best grade, large clear pieces | 83%+ clear |
| Select | One FAS face, lower grade back | 83%+ one side |
| #1 Common | Good for shorter clear pieces | 66%+ clear |
| #2 Common | Shorter/narrower clear pieces | 50%+ clear |

For cabinetry: Select or #1 Common is usually the best value. FAS is often overkill since parts are cut into shorter pieces anyway.

## Tips
- **Acclimate lumber:** Bring wood into the shop 1-2 weeks before working it. Wood moisture should match the environment where it will be installed (~6-8% for indoor use)
- **Grain direction:** Book-match or slip-match adjacent panels for best appearance
- **Wood movement:** Flat-sawn boards move more than quarter-sawn. Plan for expansion/contraction across the grain (not along it)
- **Blotch-prone species:** Cherry, maple, birch, and pine blotch with stain. Use pre-stain conditioner or gel stain
