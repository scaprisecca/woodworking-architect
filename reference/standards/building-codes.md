# Building Code Considerations for Cabinetry

> This is a general reference, not legal advice. Always check your local building codes and jurisdiction requirements.

## Kitchen
- **GFCI outlets** required within 6' of water source
- **Dedicated circuits** for dishwasher, disposal, microwave, refrigerator
- **Vent hood:** Check local requirements for CFM and ducting
- **Gas line:** Must be accessible shut-off valve
- **Counter space requirements** may be specified in local code
- **Clearance from cooktop to combustible materials:** Check appliance specs and code

## Bathroom
- **GFCI outlets** required in all bathrooms
- **Ventilation:** Exhaust fan required (check CFM requirements for room size)
- **Water-resistant materials** in wet areas
- **Clearances:** Minimum space around toilet, vanity, tub per code

## General
- **Smoke detectors:** Don't block or relocate during installation
- **Electrical boxes:** Don't cover or block access
- **Plumbing access:** Maintain access to shut-off valves and cleanouts
- **Fire blocking:** May be required in tall vertical chases (check local code)
- **Permits:** Check if cabinet installation requires a permit in your jurisdiction (usually not for replacement in-kind, but may for new layouts affecting plumbing/electrical)
- **ADA:** If client needs accessibility compliance, review ADA guidelines in detail

## Fireplace Proximity
- **Combustible materials:** Maintain minimum distance from firebox opening per code and manufacturer specs
- **Typical minimum:** 6" from firebox opening to combustible material (verify locally)
- **Mantel clearance:** Varies by projection depth — deeper mantel needs more vertical clearance
