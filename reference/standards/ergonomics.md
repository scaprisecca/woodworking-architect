# Ergonomic Design Guidelines

## Kitchen Work Triangle
The distance between sink, stove, and refrigerator:
- Each leg: 4' to 9'
- Total perimeter: 12' to 26'
- No leg should cross through a major traffic path
- Island should not block the triangle

## Reach Zones
| Zone | Height from Floor | Use |
|------|-------------------|-----|
| Comfortable reach (standing) | 24" - 66" | Most-used items |
| Maximum reach (standing) | 15" - 80" | Occasional items |
| Easy access (no bending/reaching) | 30" - 54" | Daily-use items |
| Low storage | 0" - 24" | Heavy/bulky items, less frequent |
| High storage | 66" - 84" | Seasonal, rarely used |

## Working Heights
| Activity | Optimal Height | Notes |
|----------|---------------|-------|
| Kitchen counter (general) | 36" | Standard for 5'4" - 5'10" users |
| Kitchen counter (baking/kneading) | 32" - 34" | Lower for leverage |
| Kitchen counter (chopping) | 34" - 36" | |
| Bar height | 42" | With 24" - 26" bar stool |
| Counter height seating | 36" | With 24" counter stool |
| Desk | 28" - 30" | Standard sitting desk |
| Vanity | 32" - 36" | 36" = "comfort height" |
| Closet rod (single) | 66" - 68" | From floor |
| Closet rod (double upper) | 80" - 84" | |
| Closet rod (double lower) | 40" - 42" | |

## ADA Accessibility Guidelines
When accessibility is needed:
| Element | Requirement |
|---------|-------------|
| Counter max height | 34" |
| Knee clearance under counter | 27" H x 30" W x 19" D minimum |
| Reach range (forward) | 15" - 48" from floor |
| Reach range (side) | 15" - 54" from floor |
| Pull-out shelves preferred | Over fixed deep shelves |
| Lever handles | Instead of knobs |
| Clear floor space | 30" x 48" in front of each appliance |
| Aisle width | 36" minimum, 60" for turning |

## Seating Clearances
| Seating Type | Overhang Needed | Vertical Space (counter to seat) |
|-------------|-----------------|----------------------------------|
| Bar stool at 42" counter | 12" - 15" | 10" - 12" |
| Counter stool at 36" counter | 12" - 15" | 10" - 12" |
| Chair at 30" table | 12" | 10" - 12" |
| Width per seat | 24" minimum | 30" comfortable |

## Lighting
| Location | Recommendation |
|----------|---------------|
| Under-cabinet task lighting | LED strips, 3000K-3500K, mounted at front 1/3 of cabinet |
| Inside cabinets | LED strips or puck lights, 2700K-3000K |
| Above-cabinet accent | LED strips aimed at ceiling for indirect light |
| Bookcase/display | LED strips at top of each shelf, facing down |
