# Standard Cabinet Dimensions

Reference for industry-standard cabinet sizing. These are starting points — always adjust to fit the space and client needs.

## Base Cabinets
| Dimension | Standard | Notes |
|-----------|----------|-------|
| Height (without countertop) | 34-1/2" | Includes toe kick |
| Height (with countertop) | 36" | Standard counter height |
| Depth (carcass) | 24" | 23-1/4" actual carcass + 3/4" face frame |
| Depth (with countertop overhang) | 25" - 25-1/2" | 1" - 1-1/2" overhang |
| Widths | 9", 12", 15", 18", 21", 24", 27", 30", 33", 36" | In 3" increments |
| Toe kick height | 4-1/2" | |
| Toe kick depth | 3" | Recessed from face |

## Wall Cabinets
| Dimension | Standard | Notes |
|-----------|----------|-------|
| Heights | 12", 15", 18", 24", 30", 36", 42" | 30" and 36" most common |
| Depth | 12" | 11-1/4" actual carcass + 3/4" face frame |
| Widths | 9", 12", 15", 18", 21", 24", 27", 30", 33", 36" | Match base widths |
| Bottom of wall cab to countertop | 18" | Standard backsplash space |
| Bottom of wall cab from floor | 54" | 36" counter + 18" backsplash |

## Tall / Pantry Cabinets
| Dimension | Standard | Notes |
|-----------|----------|-------|
| Heights | 84", 90", 96" | 84" common, 96" for floor-to-ceiling |
| Depth | 24" (base depth) or 12" (wall depth) | Depends on location |
| Widths | 18", 24", 30", 36" | |

## Drawers
| Dimension | Standard | Notes |
|-----------|----------|-------|
| Minimum width | 12" | Narrower gets impractical |
| Maximum width | 36" | Wider needs center slide or heavy-duty slides |
| Height (small) | 4" - 6" | Utensils, flatware |
| Height (medium) | 8" - 10" | Folded items, mixing bowls |
| Height (deep) | 12" | Pots, pans, stand mixer |
| Front overlap | 1/2" per side | Full overlay |
| Side clearance | 1/2" per side | For drawer slides |

## Doors
| Style | Overlay | Notes |
|-------|---------|-------|
| Full overlay | 1/2" overlap on all sides | Most common modern look |
| Half overlay | Covers half the face frame | Used when doors meet at a shared stile |
| Inset | Flush with face frame | Traditional look, tighter tolerances |
| Reveal between doors | 1/8" | Standard gap |

## Face Frame
| Dimension | Standard | Notes |
|-----------|----------|-------|
| Stile width | 1-1/2" | Can go to 2" for end panels |
| Rail width | 1-1/2" | Top and bottom |
| Material thickness | 3/4" | Hardwood |
| Overhang inside carcass | 1/4" | For hinge clearance |

## Shelving
| Dimension | Standard | Notes |
|-----------|----------|-------|
| Thickness | 3/4" | |
| Max unsupported span | 30" | For 3/4" plywood without sag |
| Depth | 1/2" less than carcass depth | For clearance |
| Pin hole spacing | 32mm (1-1/4") | European system standard |
| Pin hole setback from front/back | 37mm (1-7/16") | European system |

## Countertop
| Dimension | Standard | Notes |
|-----------|----------|-------|
| Thickness | 1-1/2" (stone), 3/4" (laminate) | |
| Front overhang | 1" - 1-1/2" | |
| Side overhang (at open end) | 1" | |
| Backsplash height | 4" (standard) or full (to wall cabs) | |
| Island overhang for seating | 12" - 15" | Needs support brackets |

## Common Filler Sizes
| Location | Width | Notes |
|----------|-------|-------|
| Wall filler | 1-1/2" to 6" | Between cabinet and wall |
| Corner filler | 3" | In blind corner setups |
| Appliance filler | As needed | Usually 1/4" - 1" |

## Clearances
| Situation | Minimum | Preferred |
|-----------|---------|-----------|
| Between opposing cabinets | 36" | 42" - 48" |
| Between island and perimeter | 36" | 42" |
| In front of dishwasher | 36" | |
| In front of oven | 36" | |
| Above cooktop to hood | 24" (electric), 30" (gas) | Check hood specs |
